# Knowledge Grapher

![Project Banner]("./assets/image.png") <!-- Add your project banner screenshot here -->

**Knowledge Grapher** is an AI-powered NLP Text-to-Entity Knowledge Graph Visualizer. It takes raw, unstructured text inputs, automatically extracts key entities and their relationships using Natural Language Processing, and transforms them into interactive, visual knowledge graphs.

This application is designed to help users quickly digest complex information, understand core subjects, and map out connections within large bodies of text effortlessly.

## ✨ Key Features
- **Automated Extraction:** Leverages advanced AI models (via Groq API) to accurately identify entities (e.g., people, organizations, concepts) and define how they relate to one another.
- **Interactive Visualization:** Renders dynamic, node-based graphs. Users can drag nodes, zoom in/out, and explore the web of information interactively.
- **History & Management:** Saves past text analyses securely, allowing users to revisit, manage, and compare previously generated knowledge graphs.
- **Secure Access:** Built-in authentication ensures that user data and generated graphs remain private and personalized.

## 🛠 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **AI/NLP Engine:** Groq LLM API integration
- **Visuals:** HTML Canvas / Graphing Libraries
